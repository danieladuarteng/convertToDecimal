# Convert to Decimal


The objective of this project is to convert the numbers of a X system that we use to map its values ​​to Roman system and thereby convert them into decimals.
Table of translation:

X System   | Roman System
---------  | ------------
kil        | I
jin        | V
pol        | X
kilow      | L
jij        | C
jinjin     | D
polsx      | M
